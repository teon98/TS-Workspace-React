# 개요
> 실전코드로 배우는 TS 필수 개념 React편을 수강하고 날짜별 기분 온도를 올리면 그래프로 표시되는 기능을 간단하게 과제 형식으로 구현해봤습니다. 강의에서 다루지 않았지만 그래프에 뿌려줄 데이터를 전역으로 관리하고자 Redux를 추가 사용하였습니다.
